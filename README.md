# System Design Repository

A collection of system design artifacts, diagrams, and documentation for architecture discussions and reference.

## 🗂 Structure

- `mobi/`  
  PlantUML diagrams describing core system flows (e.g., order creation, deduplication).

- `navi/`  
  *(Empty / To be documented)*

- `out/`  
  Auto-generated outputs (e.g., SVGs) from diagram sources.

- `system-design/`  
  *(Empty / To be documented)*

## 📝 Artifacts

- **UML Diagrams:**  
  - `mobi/create-order.puml`  
    Sequence/flow diagram for order creation.
  - `mobi/duplicate-order.puml`  
    Sequence/flow diagram for deduplication logic.
  - Rendered diagrams in `out/mobi/duplicate-order/`.

## 🔧 How to Contribute

1. Edit or add PlantUML files in the relevant subfolder.
2. Generate SVG/PNG outputs and place them in `out/`.
3. (Optional) Add markdown or other documentation as needed.
